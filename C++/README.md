# C++

## RF24 library

```shell
git clone https://github.com/nRF24/RF24.git
cd lib/RF24/
make all && sudo make install
```

- Installing Libs to **/usr/local/lib** for name `librf24.so.1.3.0`
- Installing Headers to **/usr/local/include/RF24**

``` c++
#include <RF24/nRF24L01.h>
#include <RF24/RF24.h>
```
